# 📊 ZOMATO EDA Project

This project performs Exploratory Data Analysis (EDA) on the**Zomato Bangalore Restaurants dataset** using Python and the following libraries:
- pandas
- numpy
- matplotlib
- seaborn

The aim is to analyze restaurant data, study customer preferences, and visualize important trends.
---

## 📂 Dataset Info
- The dataset is not included in the repository due to size limitations. please download it from link given below and update the path in the notebook before running.
- The dataset is too large for GitHub (~547 MB). You can download it from:
- Source: [Kaggle - Zomato Bangalore Restaurants Dataset](https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants)
- - File used: `zomato.csv` (cleaned version saved as `zomato_cleaned.csv`)  


---

## ✅ Objectives

- Clean and preprocess the data  
- Analyze restaurant types, cuisines, and ratings  
- Explore online order vs table booking trends  
- Study price ranges and their relation with ratings  
- Visualize insights with charts  

---
